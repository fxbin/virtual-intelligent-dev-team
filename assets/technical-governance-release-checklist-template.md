# Technical Governance Release Checklist

## Preconditions

- [ ] Scope is locked
- [ ] Risk level is declared
- [ ] Rollback trigger is defined

## Verification

- [ ] Pre-change checks passed
- [ ] Post-change checks defined
- [ ] Owner is assigned

## Delivery

- [ ] Branch / PR flow is defined
- [ ] Review gate is defined
- [ ] Release gate or hold condition is defined

## Follow-up

- [ ] Audit or retro path is defined
- [ ] Resume anchor is recorded
